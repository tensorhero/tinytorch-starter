# TinyTorch Starter Code

TinyTorch 课程起始代码——从零实现一个教学版 PyTorch。

## 快速开始

1. **克隆代码（选择你的语言分支）**

```bash
# Java
git clone -b java https://github.com/tensorhero/tinytorch-starter
# Python
git clone -b python https://github.com/tensorhero/tinytorch-starter
```

2. **核心文件**

| 语言   | 分支     | 核心文件                                             |
| ------ | -------- | ---------------------------------------------------- |
| Java   | `java`   | `src/main/java/dev/tensorhero/tinytorch/Tensor.java` |
| Python | `python` | `tinytorch/tensor.py`                                |

3. **完成实现**

在核心文件中找到 `TODO` 注释，按 stage 顺序逐步实现。

4. **本地测试**

查看 [tinytorch-tester](https://github.com/tensorhero/tinytorch-tester) 了解如何测试你的代码。
