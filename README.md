# TinyTorch Starter — Python

TinyTorch 课程起始代码（Python）——从零实现一个教学版 PyTorch。

## 结构

```
tinytorch/         # 核心实现（tensor, function, layer 等）
tinynum/           # TinyNum 依赖
tests/             # 每个 stage 一个测试 (test_e01 … test_e10)
tensorhero.yml     # 课程元数据
```
